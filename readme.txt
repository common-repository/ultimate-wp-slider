== Description ==
With Ultimate WP Slider make a slider using your own original content as each slide's content. You can design stunning, touch-enabled, responsive sliders using this plugin.

* Create slides with your own contents,
* Use the shortcode [ultimate_wp_slider] to display the slider in any page/post/widget,
* Change the style of slider from Ultimate WP Slider Options page
* To display custom title for the slider use shortcode 
   [ultimate_wp_slider]title[/ultimate_wp_slider]
* To display specific slides add the id of the slides you want to display with shortcode
   [ultimate_wp_slider id='38,39']title[/ultimate_wp_slider]
* Also you can order the slides in random order with shortcode [ultimate_wp_slider id='38,39' orderby='rand']title[/ultimate_wp_slider]


== Installation ==
1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `ultimate-wp-slider.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard
